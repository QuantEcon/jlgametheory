# jlgametheory

Python interface to GameTheory.jl

`jlgametheory` is a Python package that allows passing a `NormalFormGame` instance from `quantecon.game_theory` to `GameTheory.jl` functions via `juliacall`.

## Installation

```bash
pip install jlgametheory
```

## Dependencies

- numpy
- quantecon
- juliacall
